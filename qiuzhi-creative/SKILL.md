# qiuzhi-creative

## 元信息
name: qiuzhi-creative
description: 为秋芝餐厅生成符合品牌调性的物料设计创意，可根据用户需求生成创意描述或实际图像。

## 你的任务
当用户说要做某种物料（海报、易拉宝、包装盒等），你需要输出符合**秋芝餐厅品牌风格**的物料的设计创意。

### 指令
1. 查看 `references/format.md` → 确保输出符合标准格式规范
2. 查看 `assets/` 下参考图 → 理解品牌视觉风格（薄荷绿、3D卡通萌虾IP等）
3. 执行 `scripts/generate_image.py` → 如用户需要生成图片，则调用图像生成脚本
4. 验证输出是否符合品牌规范 → 确保包含品牌元素（如薄荷绿#5DDEB5、吉祥物等）

## 输出格式
### 创意主题
（一句话概括这个设计的核心概念）

### 视觉风格
（整体氛围、配色方向、质感）

### 画面构成
（主体元素、背景、装饰元素的布局描述）

### 细节建议
（一些能让设计更出彩的小细节）

## 资源引用
- 品牌主色：薄荷绿 `#5DDEB5`
- IP形象：3D薄荷绿卡通萌虾（大眼睛、微笑、白色条纹）
- 风格定位：3D卡通、清新时尚、年轻活力
